Languages used: C++

In this class, the object has a default and a parameterized constructor, and an overloaded printing operator. The second part's objective was to create the data structure Collection in a class called Collection and have that class hold the Stress_ball objects. This Collection class had both default and parameterized constructor, copy constructor and assignment, move constructor and assignment, destructor, and overloaded operators along with some methods such as insert, contains, remove, is_empty, total_items, print_items, and a sorting function. Finally, the third part's objective was to create a class for a new object called Jeans (very similar to Stress_ball but does have differing characteristics) and a templated Collection class (basically the Collection class but now can work with any kind of object with a set amount of parameters). This templated Collection class will now work like the Collection data structure but will function properly with Jean and Stress_ball objects.

This class included a node structure that could hold data and had next and previous pointers to other nodes in the data structure, the Big Five (copy constructor, copy assignment, move constructor, move assignment, destructor), constructors, and methods that belong to the data structure. The second part's objective was to implement the Doubly Linked List data structure but to have it templated. It was the same as the program in the first part but just now templated. Assignment 3: This assignment's objective was to implement the Binary Search Tree data structure that takes in data to create the nodes for the trees. The assignment also included the Big Five, being able to show off the time it took to search for the node, determining the average search cost for the tree, and the program had to work for perfect, random, and linear binary trees. Assignment 5: This assignment's objective was to implement a directed acyclic graph data structure. The first part consisted of creating a Vertex structure (which is the Node in this case) that would have an integer label, creating a Graph class that would have a vector of all of the Vertex objects and a vector of linked lists of integer pointers for all of the Vertex objects connecting to a Vertex object. The Graph class also included a constructor, destructor, display, overloaded operator, and build function and took in the information for the Vertex objects through a file. The second part consisted of implementing the topological sorting algorithm for the Graph class. This included creating a vector for the topological order of a graph, printing the topological order of a graph, and the actual algorithm itself.
